# 🚦 Smart Traffic Optimization and Graph Analysis System

An advanced **Design and Analysis of Algorithms (DAA)** project that demonstrates intelligent traffic management using graph algorithms, real-world maps, traffic simulation, and route optimization. The system integrates multiple shortest-path and optimization algorithms to simulate realistic traffic conditions and analyze algorithm performance.

---

## 📌 Project Overview

The Smart Traffic Optimization and Graph Analysis System models cities and roads as a **weighted graph**, where:

- 📍 Cities/Intersections → Nodes
- 🛣️ Roads → Weighted Edges

The project computes optimal routes by considering:

- Distance
- Traffic Congestion
- Weather Conditions
- Elevation
- Road Closures

It combines classical graph algorithms with modern visualization techniques to provide an interactive traffic simulation platform.
